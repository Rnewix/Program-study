Django

python -m django --version

django-admin startproject Project_Name                  Crear proyecto (carpeta)    
python manage.py startapp App_name                      Crear aplicaciones en el proyecto

Iniciar un Proyecto
    1   Crear proyecto                          django-admin startproject myproject  
    2   Inicializar BD                          python manage.py makemigrations 
                                                python manage.py migrate
    2.  (Opcional)          
            1- Cambio BD                        (En settings.py)
	        2- Cambio Time_zone                 (En settings.py) 
    2.. Checar buen funcionamiento de la APP    python manage.py check App_name 


Creacion de Aplicaciones
    1   Crear aplicacion                        python manage.py startapp App_name     
    2   Crear Carpetas y archivos faltantes
            1- urls.py 
            2- Mapeador Url                     (En APP/urls.py)
                                                    from django.urls import path
            3- Carpeta templates
    3   Registrar app al proyecto               (En settings.py)
                                                    "AppName.apps.AppNameConfig"

                                                    INSTALLED_APPS = [
                                                        'django.contrib.admin',
                                                        'django.contrib.auth',
                                                        ----Aqui-----
                                                    ]
    4   Conectar Mapeadores URL.    
        URLs apps <-> URL proyecto              Include
                                                (En Project/url.py)
                                                    from django.contrib import admin
                                                    from django.urls import path, include

                                                    urlpatterns = [
                                                        path('admin/', admin.site.urls),
                                                        path('', include('AppName.urls'))
                                                    ]

Creacion de modelos
    1   Creacion del modelo                     (En APP/Models.py) 
    2   Importar modelos                        (En APP/admin.py)
                                                    from AppName.models import model_1, model_2
    3   Registrar modelos                       (En APP/admin.py)
                                                    admin.site.register(Model_1)
                                                    admin.site.register(Model_2)
